# QA Automation Framework

A scalable Selenium Automation Framework developed using Java, Selenium WebDriver, TestNG, Maven, and the Page Object Model (POM). This project is designed following industry best practices and is being enhanced with enterprise-level automation features.

## Tech Stack

- Java 17
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- Git & GitHub

## Features Implemented

- Driver Factory
- Base Test
- Base Page
- Config Reader
- Wait Utilities
- Screenshot Utility
- Page Factory
- Login Test Automation
- Maven Project Structure
- Git Version Control

## Project Structure


## Prerequisites

- Java 17+
- Maven
- Git
- Eclipse or IntelliJ IDEA
- Google Chrome

## How to Run

1. Clone the repository
2. Open it as a Maven project
3. Update Maven dependencies
4. Run `LoginTest` as a TestNG test
